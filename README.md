# Website Uptime Monitor

## Description
A Python-based tool that checks whether a website is online and measures its response time.

## Features
- Website availability check
- Response time calculation
- Simple CLI-based monitoring tool

## Technologies Used
- Python
- Requests library

## How to Run
1. Install dependency:
   pip install requests
2. Run the script:
   python monitor.py
3. Enter website URL

## Use Case
Used to monitor website availability in DevOps and cloud environments.
